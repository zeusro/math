## No best policy in n

The global optimal solution and the local optimal solution are relative.

Due to interference in the n+1th dimension, a global optimum does not exist.

Prove:

### 1

  n < n + 1

### 2

$\not\exists x^* \in S,\ \forall x \in S,\ f(x^*) \leq f(x)$
